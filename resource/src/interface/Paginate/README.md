# Paginate.js
Add class `@paginate` to the container element.

Parameters are defined by liquid properties. 

```html
<div class="@paginate">
    <!-- Content -->
</div>
```

Class | Functionality
------------ | -------------
`@paginate` | Creates an instance of Paginate.js
`@btn-prev` | Routes to the previous
`@btn-next` | Routes to the next page
