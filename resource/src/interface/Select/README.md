# Select.js
Add class `@select` to the container element of the number input element.

Parameters are defined by datatypes and native input properties. 

```html
<div class="@select">
    <!-- Content -->
</div>
```

Class | Functionality
------------ | -------------
`@select` | Creates an instance of Select.js
